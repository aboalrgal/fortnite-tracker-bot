\# Fortnite Tracker Bot



A Discord bot that tracks \*\*every update\*\* from Fortnite-API:

\- Cosmetics

\- Shop

\- News

\- Map

\- Playlists

\- AES keys



The bot automatically compares previous data with new API responses every 5 minutes  

and sends update notifications to a Discord channel.



\## Setup



1\. Add your Discord bot token in Railway variables:

&nbsp;  TOKEN=<your\_token>

&nbsp;  CHANNEL\_ID=<your\_channel\_id>



2\. Deploy to Railway.



3\. Bot runs 24/7 and scans Fortnite API for any new changes.
